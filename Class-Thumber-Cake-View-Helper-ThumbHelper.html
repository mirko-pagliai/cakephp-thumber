__call()
__call(string $name, array $params)

Magic method. It dynamically calls all other methods. Each method takes these arguments:

  • $path Path of the image from which to create the thumbnail. It can be a relative path (to APP/webroot/img), a full path or a remote url;
  • $params Parameters for creating the thumbnail;
  • $options Array of HTML attributes for the img element.
